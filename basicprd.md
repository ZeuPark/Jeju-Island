프로젝트 개요
이 프로젝트는 제주 HVDC 연계선 수전 전력 데이터를 사용해 하루와 일별 패턴을 빠르게 관찰하고 계통 제약과 출력 제한 가능성을 직관적으로 설명하는 것을 목표로 한다. 예측 모델이나 복잡한 알고리즘 없이 시각화와 해석만으로 의미를 전달한다.

문제 정의
제주는 신재생 발전 비중이 높아 특정 시점에 발전량이 수요와 계통 수용 능력을 초과한다. 이때 HVDC 연계선 수전 전력의 시간대별 패턴과 일별 변동을 이해하지 않으면 출력 제한과 셧다운이 왜 발생하는지 설명하기 어렵다.

목표
하나의 공공 데이터셋으로 제주 HVDC 수전 전력의 시간대 패턴과 일별 변동성을 한눈에 보여준다.
계통 제약과 출력 제한 이슈를 데이터 기반 문장으로 설명한다.
1시간 내 완성 가능한 최소 프로젝트로 실행 부담을 없앤다.

비목표
전력 수요 예측이나 발전량 예측 모델을 만들지 않는다.
딥러닝이나 복잡한 통계 분석을 포함하지 않는다.
실시간 시스템이나 웹 서비스로 확장하지 않는다.

대상 사용자
에너지 전력 도메인에 관심 있는 데이터 분석가
전력 계통 이슈를 비전문가에게 설명해야 하는 엔지니어
빠른 포트폴리오 스냅샷이 필요한 학생

데이터
한국전력거래소 공개 데이터
제주 일별 시간대별 HVDC 연계선 수전 전력
보조 맥락 데이터로 전일 전력수급실적 페이지와 관련 기사 참고

핵심 질문
하루 동안 HVDC 수전 전력은 어떤 시간대 패턴을 가지는가
날짜별로 이 패턴은 얼마나 변동하는가
이 변동은 계통 제약과 출력 제한 논의와 어떻게 연결되는가

기능 요구사항
CSV 파일 로드
날짜와 시간대 컬럼 파싱
시간대별 수전 전력 선 그래프 생성
여러 날짜의 시간대 패턴 겹쳐 보기
일별 총 수전 전력 요약 그래프 생성

비기능 요구사항
전체 작업 시간 1시간 이내
노트북 한 개로 실행 가능
외부 API 호출 없음

산출물
Jupyter Notebook 1개
그래프 3개 이상
결론 요약 텍스트 5줄 내외

성공 기준
그래프만 보고도 하루 패턴과 변동성이 직관적으로 보인다.
출력 제한과 계통 병목에 대한 설명이 데이터와 연결된다.
추가 예측 없이도 프로젝트가 완결된 느낌을 준다.

리스크와 대응
데이터 컬럼명이 예상과 다를 수 있음
대응 컬럼 매핑을 코드 상단에서 명시적으로 처리

확장 아이디어
월별 계절 비교
전일 전력수급실적과 HVDC 비중 비교
날씨 데이터와의 간단한 상관 관찰

한 줄 요약
이 프로젝트는 제주 HVDC 수전 전력 데이터를 통해 왜 전기가 남아도 셧다운이 발생하는지 구조적으로 설명하는 1시간 관찰 실험이다.
